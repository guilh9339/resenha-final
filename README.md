#  Resenha 

Projeto desenvolvido para a disciplina de Interface Humano Computador. A aplicação é uma interface interativa para consulta de resultados, estatísticas e escalações de jogos de futebol, baseada em um protótipo fiel criado no Figma.

##  Links
- **Repositório:** https://github.com/guilh9339/resenha-final

##  Tecnologias Utilizadas
- **HTML5:** Estrutura semântica e organização de conteúdo.
- **CSS3:** Estilização avançada, variáveis (root), Flexbox, Grid e Design Responsivo (Mobile First).
- **JavaScript (ES6+):** Lógica de autenticação, manipulação de DOM, sistema de busca dinâmica e navegação entre abas.
- **Firebase:** Backend as a Service (BaaS) utilizado para Autenticação de usuários e Banco de Dados (Firestore).

##  Funcionalidades
1.  **Sistema de Login e Cadastro:** Autenticação real conectada ao Firebase.
2.  **Busca Dinâmica:** Pesquisa por times ou apelidos (ex: "Timão", "Verdão") com redirecionamento inteligente.
3.  **Navegação por Abas:** Alternância de conteúdo (Tira-teima, Visão Geral, Escalações, Tabela) sem recarregar a página.
4.  **Design Fiel:** Reprodução pixel-perfect do protótipo em modo Dark/Neon.
5.  **Responsividade:** Layout adaptável para dispositivos móveis.
